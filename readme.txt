Este programa manipula una lista de personas con diferentes campos para cada persona usando la estructura de arreglo, 
cada una de las tareas están especificadas en el menú del programa una vez ejecutado

INSTRUCCIONES PARA EJECUTAR PROGRAMA

LINUX:
	COMNADOS:
	gcc -o programa programa.c
	./programa

WINDOWS
	Debes de tener un compilador C instalado, puedes obtener uno en  https://mingw-w64.org/doku.php. Sigue las
	instrucciones de instalacion.
	
	COMANDOS:
	gcc programa.c -o programa
	programa.exe
